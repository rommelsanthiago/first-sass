## Instalação
Execute a linha de comando abaixo para clonar o repositório:
```
    git clone https://github.com/rommelsanthiago/first-sass.git
```
No console do terminal execute o comando:
```
    npm install
```
Para executar a compilação do ``scss`` execute:
```
    gulp
```
